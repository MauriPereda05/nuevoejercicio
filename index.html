HTML FUNDAMENTOS

Estructura base HTML

La estructura base HTML son elementos basicos que se utilizan en una pagina web HTML los elementos que se usan son DOCTYPE,html,head,body,header.

Comentarios

Los comentarios en HTML son texto que no sera interpretado en el navegador,cualquier texto que este encapsulado en ! ---- y ---- no seran interpretados en la pagina.

Atributos

Los atributos en HTML pueden ser utilizados para asignar valores a los campos adicionales a los elementos,los atributos estan divididos en 2 partes

Obligatorios

Los atributos obligatorios en HTML son aquellos atributos que son necesarios para la estructura y funcion de un elemento,por ejemplo,el atributo alt es obligatorio en el elemento img para especificar una descripcion alternativa de la imagen.

Opcionales

Los atributos opcionales en HTML son aquellos que se pueden o no especificar dependiendo de las necesidades especificas del usuario.

HTML ETIQUETAS

Encabezados

Los encabezados se utilizan para determinar la importancia de un titulo en la pagina web.

Parrafos

Los parrafos en HTML es texto distribuido en parrafos.

Enlaces

Los enlaces se utilizan para que los usuarios puedan navegar a traves de la pagina web y otras paginas en el sitio o en otros sitios web.
