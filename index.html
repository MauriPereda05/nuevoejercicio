HTML FUNDAMENTOS

Estructura base HTML

La estructura base HTML son elementos basicos que se utilizan en una pagina web HTML por ejemplo, los elementos que se usan son DOCTYPE,html,head,body,header.

Comentarios

Los comentarios en HTML son texto que no sera interpretado en el navegador,cualquier texto que este encapsulado como por ejemplo < ! ---- y ---- > no seran interpretados en la pagina.

Tipo de etiquetas

Las etiquetas en HTML estan divididas en 4 partes

Apertura y Cierre

Un pequeño ejemplo sobre la apertura y cierre en HTML de un párrafo de texto:se utiliza la etiqueta < p> para iniciar el párrafo y termina con un cierre de etiqueta en < /p> esto hace una apertura y un cierre.

Self closing

Es casi lo mismo como en apertura y cierre solo que en este no tiene cierre por ejemplo < img src="" alt=""> no tiene una cierre y a esto se lo llama self closing.

Bloque

La etiqueta de Bloque en HTML son los que ocupan un espacio horizontal en el sentido de que estos elementos de la pagina tienen una estructura interna que indica el navegador como se deben mostrar como por ejemplo:

Inline

En HTML los inline solo ocupan su contenido ya que no tienen una estructura de inicio y cierre que determina su tamaño o forma como por ejemplo.

Hola Mundo

Los inline usando < span> < /span> solo ocupan su contenido,excepto los encabezados " < h1>" los parrafos "< p>" y los div

Atributos

Los atributos en HTML pueden ser utilizados para asignar valores a los campos adicionales a los elementos,los atributos estan divididos en 2 partes

Obligatorios

Los atributos obligatorios en HTML son aquellos atributos que son necesarios para la estructura y la funcion de un elemento,por ejemplo,el atributo alt es obligatorio en el elemento < img> para especificar una descripcion alternativa de la imagen.

Opcionales

Los atributos opcionales en HTML son aquellos que se pueden o no especificar dependiendo de las necesidades especificas del usuario como por ejemplo: < h1 class="" > este es otro tipo de atributo y no es obligatorio ponerlo ya que class se usa para CSS

Enlaces a paginas webs externas

Los enlaces a paginas webs externas son las que abre una nueva pestaña sin interrumpir la pestaña actual con la ayuda de target="_blank" como por ejemplo < a href="https://www.google.com/" target="_blank"> < /a> de lo contrario si un enlace no lleva target="_blank" el navegador cargará la pagina vinculada en la misma ventana o pestaña en la que se encuentra el usuario actualmente.

HTML5 ETIQUETAS

Header

El header es el contenido introductorio de su ancestro mas cercano,se utiliza para definir la cabecera de un documento o de una seccion dentro del mismo.Generalmente se encuentra al principio de una pagina y puede contener elementos como un titulo

Nav

La etiqueta nav sirve para crear menus de navegacion en HTML por ejemplo:

< nav>
< ul>
< a href="#">inicios< /a>
< /ul>
< /nav>

En este codigo < nav> se esta utilizando para definir la seccion de navegacion del sitio web,lo que ayuda a los usuarios y a los motores de busqueda a entender la estructura y la jerarquia del contenido.

Footer

El footer es el pie de pagina de una seccion o contenido.El pie de pagina generalmente contiene informacion como por ejemplo,enlaces adicionales,derechos autor,creditos y otra informacion relevante para el sitio web

Section

El section es una seccion o division de contenido,en HTML es un elemento que se utiliza para agrupar y organizar contenido relacionado dentro de una pagina web por ejemplo si estuvieramos diseñando una pagina web para un blog de viaje podriamos usar < section> para organizar el contenido.

Article

El section es una seccion o division de contenido,en HTML es un elemento que se utiliza para agrupar y organizar contenido relacionado dentro de una pagina web por ejemplo si estuvieramos diseñando una pagina web para un blog de viaje podriamos usar < section> para organizar el contenido.

Main

Es el contenido principal de la pagina UNICO en HTML5 es considerado unico porque esta destinado a contener el contenido principal y unico de una pagina web como por ejemplo

< main>
< p>Bienvenido a mi página web. Aquí encontrarás información sobre diversos temas.


< p>¡Espero que disfrutes tu visita!


< /main>

< main> contiene simplemente un parrafo de bienvenida que da una breve introduccion a la pagina web.Esta estructura ayuda a organizar y presentar el contenido de manera clara y semantica

Aside

Contenido no necesariamente relacionado,El uso mas comun de < aside> es para elementos como barras laterales,notas al margen o anuncios.
